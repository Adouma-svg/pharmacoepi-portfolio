The Confounding Problem in RWE

In randomised trials, treatment assignment is random. In RWE, physicians choose treatment based on patient characteristics — sicker patients were more likely to receive FOLFIRI here. A naive Kaplan-Meier is therefore biased.

Step 1

Estimate PS

Model probability of FOLFOX given baseline covariates using logistic regression

Step 2

Compute Weights

IPTW = 1/PS for treated; 1/(1−PS) for controls. Creates pseudo-population with balanced covariates.

Step 3

Assess Balance

Love plot shows SMD before/after weighting. Goal: all |SMD| < 0.10.

# IPTW using WeightIt — ATE estimand
library(WeightIt); library(cobalt); library(survival)

ps_model <- weightit(
  treatment ~ age + sex + ecog + cci + primary_site +
    kras + braf + liver_met + n_met_sites,
  data     = mCRC,
  method   = "ps",
  estimand = "ATE"
)

# Balance check — standardized mean differences
love.plot(ps_model, threshold = 0.1)

# Weighted Cox model (robust sandwich SE)
cox_iptw <- coxph(
  Surv(os_months, os_event) ~ treatment,
  data    = mCRC,
  weights = ps_model$weights,
  robust  = TRUE
)
tidy(cox_iptw, exponentiate = TRUE, conf.int = TRUE)
/* SAS — Propensity Score + Weighted Cox */
proc logistic data=mcrc descending;
  class treatment sex kras braf primary_site / param=ref;
  model trt_folfox = age sex ecog cci primary_site kras braf
                     liver_met n_met_sites;
  output out=ps_out p=ps_hat;
run;

data ps_weighted;
  set ps_out;
  if treatment="FOLFOX" then sw_iptw = 0.5 / ps_hat;
  else sw_iptw = 0.5 / (1 - ps_hat);
run;

proc phreg data=ps_weighted covs(aggregate);
  class treatment (ref="FOLFIRI") / param=ref;
  model os_months * os_event(0) = treatment / ties=efron rl;
  weight sw_iptw; id patient_id;
run;

Why These Design Choices Matter

New-User Design

Restricts to patients initiating therapy. Eliminates depletion-of-susceptibles bias (prevalent users have already tolerated the drug) and immortal time bias.

Active Comparator

Comparing to endocrine monotherapy (same indication) rather than "no treatment" minimises confounding by indication and channelling bias.

Negative Control Outcome

Urinary tract infection has no biological link to CDK4/6 inhibitors. A non-null result would flag unmeasured confounding. A null result increases confidence in study validity.

Exact Poisson CI

Normal approximation CIs fail with small event counts. Exact Poisson CIs (chi-square method) are used for reliable interval estimation.

# Exact Poisson incidence rates per 100 person-years
ir_table <- breast_ca |>
  group_by(treatment) |>
  summarise(
    n_events = sum(cv_event),
    py       = sum(person_years),
    ir       = n_events / py * 100,
    ir_lower = qchisq(0.025, 2 * n_events)     / (2 * py) * 100,
    ir_upper = qchisq(0.975, 2 * (n_events + 1)) / (2 * py) * 100
  )

# Poisson regression — IRRs
poisson_fit <- glm(
  cv_event ~ treatment + age + htn + prior_cvd + cci +
    offset(log(pmax(person_years, 0.01))),
  data   = breast_ca,
  family = poisson(link = "log")
)

# Negative control outcome — UTI
uti_fit <- glm(

    offset(log(pmax(person_years, 0.01))),
  data = breast_ca, family = poisson()
